# On Ona (Дуэт)

**Дуэт** — игра для двоих на одном телефоне: для пар, которые хотят узнать друг друга ещё лучше и весело провести вечер вместе.

Никаких аккаунтов и загрузок — открыл ссылку, ввёл имена и сразу играешь. Вопросы звучат вслух по очереди, а половинка честно оценивает ответ; в совместных режимах пара отвечает вместе и зарабатывает общие очки.

- 💗 **«Узнай меня»** — вопросы о партнёре по очереди, три уровня сложности (от «какой любимый цвет» до вопросов на глубокое доверие) плюс отдельная категория «Романтика 18+»
- 🏆 **«Викторина вдвоём»** — 380+ вопросов в 8 темах (кино, музыка, еда, мир, наука, спорт, любовь, разное), 4 варианта ответа и огненные бонус-раунды ×2 с таймером
- 👉 **«Кто из вас?»** — весёлые вопросы без правильных ответов, просто показывайте друг на друга
- 🎭 Маскоты-привидения реагируют на ответы, общий счёт пары и рекорды сохраняются на устройстве
- 📱 Устанавливается на домашний экран как приложение (PWA), работает офлайн
